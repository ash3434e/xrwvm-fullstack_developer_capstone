# fullstack_developer_capstone

This project is the final capstone for the IBM Full Stack Developer Professional Certificate.
It features a full-stack Django and React application integrated with a Cloud backend and MongoDB.

## Features
- User Authentication (Login, Logout, Sign up)
- View Dealerships across different locations
- Add reviews for dealerships with Sentiment Analysis
- Containerized using Docker
- CI/CD pipeline integrated using GitHub actions
- Hosted and deployed on Cloud
